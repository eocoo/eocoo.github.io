redis 常用操作

words: 1.4k    views:    time: 5min
日常笔记
redis

Redis(Remote Dictionary Server)，即远程字典服务，是一个开源的使用C语言编写、支持网络、可基于内存亦可持久化的日志型、Key-Value数据库，并提供多种语言的API。

redis是一个基于内存的key-value存储系统，类似Memcached，区别的是redis会周期性的把更新的数据写入磁盘（RDB）或者把修改操作写入追加的记录文件（AOF）， 并在此基础上实现master-slave(主从)同步。
因此redis可以用于高速缓存，同时它支持多种数据类型，以及集群和事务的操作，进而也可以用于发布订阅、地图信息分析、计数器等系统

1. 安装配置

Linux下安装非常简单，官网下载安装包，解压后执行：make install，安装完之后在\usr\local\bin下便可以看到redis的可执行文件

然后修改配置文件redis.conf，便可以指定配置文件进行启动：redis-server /path/to/redis.conf，当然也可以修改多个配置文件启动多个redis实例

2. 操作

使用redis-cli -h 127.0.0.1 -p 6379便可以连接对应的redis实例，然后所有的操作（原子操作）都可以通过命令来进行

db 操作：

select 3 ：切换数据库
dbsize ：查看当前库大小
flush db ：清空当前数据库
flushall ：清空所有数据库

key 操作：

keys * ：获取所有的key
exists key ：是否存在key
move key 1 ：从数据库1中删除key
type key ：查看key类型
expire key 10 ：设置key过期时间
ttl key ：查看key剩余时间

3. 数据类型

String：

set key value ：设置key的值
get key ：获取key的值
mset k1 v1 k2 v2 k3 v3 ：设置多值
mget k1 k2 k3 ：获取多值
setnx key value ：如果当前key不存在，则设置成功
msetnx k1 v1 k2 v2 ：如果当前key不存在，则设置成功（一个失败则全部失败）
getset key value ：返回当前值，并设置新值
setex key 10 value ：设置key的值，并设置10秒过期时间
strlen key ：查看key对应值的长度
append key value ：在key的值后面拼接字符串，如果key不存在则相当于set
setrange key 2 value ：设置key的值，并从位置2开始的地方进行覆盖
getrange key 0 5 ：获取key的值，并截取位置0-5
incr key ：key的值+1
decr key ：key的值-1
incrby key 10 ：key的值+10
decrby key 10 ：ey的值-10

场景：使用mset/mget存取对象，相对直接存取json可以更灵活的操作

保存人员：mset user:1:name shanhm user:1:age 30 user:1:sex man
获取人员属性：mget user:1:name user:1:age user:1:sex

List

lpush key v1 v2 ：从头部向列表中放入值
get key ：获取key的值

Set

set key value ：设置key的值
get key ：获取key的值

Hash

set key value ：设置key的值
get key ：获取key的值

Zset

set key value ：设置key的值
get key ：获取key的值

geospatial

set key value ：设置key的值
get key ：获取key的值

hyperloglog

set key value ：设置key的值
get key ：获取key的值

bitmaps

set key value ：设置key的值
get key ：获取key的值
