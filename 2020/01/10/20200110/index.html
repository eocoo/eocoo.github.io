mac 常用操作

words: 862    views:    time: 3min
日常笔记

最近拿了米娅同学mac本子在用，顺便教教她写代码，但是感觉很不习惯，估计windows用了很多年的小伙伴都有这种感觉，所以简单整理了一下mac的常用操作

1. 快捷键

macOS的快捷键与windows还是有很大差异的，首先有没有鼠标右击的概率，不过可以option + 鼠标选择点击选项，这样也说得过去吧，另外它的command键很多时候相当于windows的control键，需要适应一下

剪切、拷贝、粘贴和其他常用

option + 鼠标 右击选项
command + c/v/x 将所选内容拷贝 / 粘贴 / 剪切
command + z 撤销操作
Shift + command + z 恢复撤销的操作
command + a 全选
command + f 文稿中查找内容
command + g 文稿中查找内容的下一个位置
Shift + command + g 文稿中查找内容的上一个位置
command + h 隐藏最前面的应用窗口
option + command + h 隐藏除最前面应用之外的其它窗口
command + m 最小化最前面的应用至程序坞
command + o 打开所选项
command + s 保存当前文件
command + t 新建标签页
command + w 关闭最前面的应用窗口
option + command + w 关闭所有的应用窗口
option + command + esc 强制退出应用
command + space 搜索栏，比如输入terminal打开终端
control + command + f 切换全屏 / 窗口化
command + tab 切换打开的应用
control + command + q 锁屏
command + d 复制所选文件
option + command + d 显示 / 隐藏程序坞
command + d 复制所选文件
command + n 新建应用程序实例
command + l 直接定位浏览器地址栏
command + delete 将所选项移到废纸篓
shift + command + delete 清空废纸篓
command + k 连接服务器

2. 终端命令

macOS的终端做的确实比windows好很多，虽然现在windows也推出了自己的Windows Terminal，号称非常好用，但实际体验过之后感觉一言难尽。所以这也可能是mac让开发人员觉得更好用的一方面。
至于命令其实与Linux基本差不多，这里还是觉得如果macOS的命令能与Linux保持一致，那就更好了

目录操作

pwd 当前路径
cd 切换目录
ls 列出目录内容
dircmp 比较两个目录的内容，比如： dircmp dir1 dir2
mkdir 创建目录
rmdir 删除目录
mvdir 移动目录，比如重命名： mvdir dir1 dir2

文件操作

open 使用默认程序打开文件，比如： open filename
cp 复制文件或目录，比如： cp file1 file2
rm 删除文件或目录，比如： rm filename
mv 改变文件名或所在目录，比如： mv file1 file2
find 查找匹配文件，比如： find . -name "*.c" -print
ln 创建文件联接，比如： ln -s file1 file2

选择操作

comm 显示两有序文件的公共和非公共行，比如： comm file1 file2
wc 统计文件的字符数、词数和行数，比如： wc filename
nl 给文件加上行号，比如： nl file1 > file2
uniq 去掉文件中的重复行，比如： uniq file1 file2

其它操作

ps 查看进程列表
kill 终止进程，比如： kill -9 30142
env 查看环境变量
df 查看磁盘使用情况
du 查看目录情况，比如： du -sh * | sort -rh
top mac没有free命令，查看内存使用可以通过 top -l 1 | head -n 10 ，实在不行就打开活动监视器吧
